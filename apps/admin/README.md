## Usage

```bash
npx create-next-app -e https://github.com/templ/apps/admin
```

## Features

- Next.js 14 App Directory
- Radix UI Primitives
- Tailwind CSS
- Icons from [Lucide](https://lucide.dev)
- Dark mode with `next-themes`
- Tailwind CSS class sorting, merging and linting.
